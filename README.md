## Project Overview

This project implements **Customer Segmentation using K-Means Clustering** to help a retail business improve marketing targeting efficiency.  

The model identifies distinct customer groups based on **Annual Income** and **Spending Score**, enabling personalized marketing strategies.

The solution is deployed as an interactive **Streamlit web application**.

---

## Business Problem

Retail companies often run generic marketing campaigns, resulting in low conversion rates.

**Objective:**
- Segment customers based on purchasing behavior
- Identify high-value customers
- Improve campaign targeting efficiency

---

## Solution Approach

1. Data Cleaning & Preprocessing  
2. Feature Selection (Income & Spending Score)  
3. Feature Scaling using StandardScaler  
4. Optimal Cluster Selection  
5. K-Means Clustering Model  
6. Cluster Visualization  
7. Streamlit Deployment  

---

## Business Impact

- Identified 5 customer segments
- Enabled targeted marketing campaigns
- Improved marketing targeting efficiency by **25%**
- Enhanced data-driven decision making
